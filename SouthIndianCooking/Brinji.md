
# Brinji Recipe & Preparation

**Date:** 25/02/2023  

## Ingredients

| Name | Quantity | Remarks |
|------|----------|---------|
| Basmati Rice / Jeeraga Samba | 1 Cup | Soak 15 minutes |
| Water | 1:2 to 1:3 ratio | - |
| Coconut Milk | 1 Cup | - |
| Ghee | 4 tbsp | - |
| Bay Leaf | 1 | - |
| Star Anise | 1 | - |
| Saunf | 1 tsp | - |
| Green Chilli | 2 | Cut |
| Big Onion | 2 | Sliced |
| Ginger & Garlic | 2 tsp | - |
| Tomato | 2 | Sliced |
| Turmeric Powder | 1/2 tsp | - |
| Coriander Powder | 1 tsp | Heaped |
| Red Chilli Powder | 1 tsp | - |
| Garam Masala | 1 tsp | - |
| Saunf Powder | 1 tsp | - |
| Mint Leaves | 1 handful | - |
| Coriander Leaves | 1 handful | - |
| Carrot | 1 | - |
| Beans | 4 | - |
| Potato | 1 | - |
| Cashew | 10 | - |
| Bread Pieces | 3 Slices | - |
| Salt | 2 tsp | - |  

## Preparation

1. Use a heavy bottom pan.  
2. Heat ghee and add bay leaf, star anise, and saunf.  
3. Add sliced onions and cook until golden brown.  
4. Add ginger & garlic paste.  
5. Add tomatoes and cook well.  
6. Add turmeric, coriander powder, red chilli powder, garam masala, and saunf powder.  
7. Add mint and coriander leaves.  
8. Mix well.  
9. Add chopped vegetables (carrot, beans, potato).  
10. Add soaked rice.  
11. Add water and coconut milk.  
12. Add salt and mix well.  
13. Cook on high flame for 15–20 minutes.  
14. For dum cooking, place on tawa for 10 minutes.  
15. Garnish with fried bread pieces and coriander leaves.
